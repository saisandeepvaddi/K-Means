# K-Means

Implementation of K-Means algorithm on 2-Dimensional coordinate points.  
+ Note: This is done in IntelliJ IDEA IDE. You can run directly there.

```Java
Usage from command line:
Move to src directory

Compile : javac KMeans.java

Run : java KMeans <no-of-clusters> <input file> <output file>
Example: java KMeans 5 input.data output.data

```
